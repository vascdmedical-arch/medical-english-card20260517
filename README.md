# 診察英会話 暗記カードアプリ

日本語の診察会話から、対応する英語表現を覚えるための暗記カードアプリです。

PC・スマートフォンのブラウザで使用できます。

## 公開ページ

https://vascdmedical-arch.github.io/medical-english-cards/

## 主な機能

- 日本語のカードを表示
- カードをタップすると英語を表示
- 英語音声の再生
- 「覚えた」チェック
- 「再度試す」チェック
- 「再挑戦のみ」表示
- キーワード検索
- カードの追加
- 追加したカードやチェック状態をブラウザ内に保存
- JSON形式でデータの書き出し・読み込み

## 使い方

1. トップページを開きます。
2. 日本語のカードを見て、英語表現を考えます。
3. カードをタップすると英語が表示されます。
4. 必要に応じて「音声を再生」を押します。
5. 覚えたカードは「覚えた」にチェックします。
6. もう一度練習したいカードは「再度試す」にチェックします。

## スマートフォンで使う方法

スマートフォンのブラウザで開いたあと、共有メニューから「ホーム画面に追加」すると、アプリのように使えます。
